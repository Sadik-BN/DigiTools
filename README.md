# DigiTools
**This is a project showcasing react with javascript application. Data Fetching from JSON. Applying different types of npm Packages (Lucide, React Toastify etc). Event handling. Usage of useState() and use() hooks. And many more**

## Features:
**1. This system fetches product data from JSON using axios and show them in nice looking cards** <br/>
**2. Products can be added to cart** <br/>
**3. In Cart there is mini statement of products and their total price to procceed for payment. They can also be removed. Price will be automatically adjusted.***
